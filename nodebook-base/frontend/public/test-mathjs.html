💡 How to Use

Expression: Enter mathematical expressions using Math.js functions. You can combine multiple functions and use basic arithmetic operators (+, -, *, /, ^).

Scope: Define variables as JSON that can be used in your expressions. For example, {"x": 5, "y": 3} allows you to use x and y in your expressions.

Examples: Try expressions like "sqrt(x^2 + y^2)" or "mean([1,2,3,4,5])" to see the power of Math.js functions.
